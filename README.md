Ecopet

Projeto de site institucional minimalista desenvolvido para apresentar a Ecopet, uma proposta independente focada em produtos sustentáveis para gatos.

O objetivo do projeto é unir design simples, identidade visual ecológica e experiência de navegação limpa, refletindo a ideia de consumo consciente e produção em pequena escala.

Este projeto foi desenvolvido como experiência prática de desenvolvimento front-end e também para compor portfólio.

Sobre o projeto
A Ecopet nasce da ideia de criar produtos voltados ao universo felino com um olhar mais cuidadoso para o planeta.

O site apresenta:
- a proposta da marca
- o produto inicial (caminha para gatos)
- a possibilidade de apoiar o projeto
- a identidade visual sustentável da Ecopet

O projeto foi pensado para crescer futuramente, podendo receber novos produtos e novas páginas.
